# 台风巴威（2026）WRF模拟

用WRF-ARW对2026年第9号台风"巴威"（Bavi）登陆浙闽沿海过程的区域数值模拟。

- **起报时间**：2026-07-10 18Z
- **驱动场**：NCEP GFS 0.25°
- **Run A**：d01(15km,全国范围) + d02(5km,登陆走廊) 双层嵌套，84小时，聚焦登陆过程
- **Run B**：单层15km全国范围，240小时(10天)，聚焦登陆后延伸降雨影响

## 目录结构

- `results/` — 路径图、大风落区图、累计降雨图（含全国范围版本）、登陆过程动画、路径追踪CSV
- `scripts/` — 后处理脚本（不依赖wrf-python，直接用netCDF4手算SLP/风速/降雨）、domain核验脚本、动画生成脚本
- `namelists/` — WPS/WRF namelist配置（Run A登陆段 + Run B延伸段）

|---|---|---|
